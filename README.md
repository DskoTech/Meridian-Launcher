# Meridian-Launcher
A Customizable Pseudo Game System OS Launcher and Software Suite To Replace The Windows Desktop.
Programs: 
1.Meridian Explorer: A dual pane controller based file browser and manager so you can use Windows Explorer less.
2.Meridian Launcher: A Video Game Console Inspired Program and Script Launcher with built in system controls and shortcuts to vital windows sections. It has support for custom backgrounds start up videos and overlays. It has a built in Video gallery and video player, A built in music gallery and Mp3 Player and built in picture gallery with customizable folder assignments. there is support for Custom program catagory Sections, it features a launcher for the default system borwser and the suite's custom chromium based controler based browser.

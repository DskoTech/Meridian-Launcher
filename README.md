# Meridian-Launcher
A Customizable Pseudo Game System OS Launcher and Software Suite To Replace The Windows Desktop.

Programs: 

1.Meridian Explorer: A dual pane controller based file browser and manager so you can use Windows Explorer less.

2.Meridian Launcher: A Video Game Console Inspired Program and Script Launcher with built in system controls and shortcuts to vital windows sections.
It has support for custom backgrounds start up videos and overlays. It has a built in Video gallery and video player, A built in music gallery and Mp3 
Player and built in picture gallery with customizable folder assignments. there is support for Custom program catagory Sections, it features a launcher 
for the default system browser and the suite's custom chromium based controler based browser. 

3.onscreenmenu: A program that gives any computer easily enableable and disableable controler to mouse and virtual keyboard support. it also features
assinable shortcut keys to launch / and switch between programs / scripts for executing hot key commands and a recent tabs menu with select that lets 
you switch between open aops easily.enable disable with start and select combo and close with l3 & r3 or menu options.

4.Meridian Game Library & Meridian Exporter: A default option for Meridian Launcher's games tab,  Meridian Exporter interfaces with Playnite and makes a 
gallery of your games libraries inported to playnite, sorts from GoG Amazon Steam Epic and all Others; lets you directly open playnite to start downloads for 
any missing games.

5.CyberDeckBrowser: A controller based chromium browser with optional cyberpunk aesthetics.
